## 2024-05-18 - Input focus visibility
**Learning:** By adding focus-within styles (`focus-within:ring-2 focus-within:ring-amber-500 focus-within:ring-offset-2`) to containers of outline-none inputs, we can maintain custom focus indicators without relying on default browser outlines, improving keyboard accessibility.
**Action:** Use focus-within on container elements to provide clear visual feedback when their inner outline-none inputs receive focus.
